Grids . Documentation

À titre expérimental, les grilles du framework utilisent les modules CSS Grid Layout, Container Queries et les opérateurs logiques (Media Queries Level 4). En raison du support limité de @container, y compris pour des navigateurs récents, un polyfill JS/CSS faisant appel à une feuille de style plus classique (mais toujours avec grid layout) est prévu pour palier à cette situation provisoire.
